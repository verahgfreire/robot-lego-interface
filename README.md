# Lego Mindstorms Robot Remote Interface

Remote interface, developed in Java Swing, to control the behavior of the Lego Mindstorms robot.
Asynchronous development using Java Threads, semaphores and monitors.

## Developed behaviors

* Avoid obstacles
* Wander
* Follow wall
